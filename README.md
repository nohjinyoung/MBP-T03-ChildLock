> **차량 내 영유아 탑승객의 절대적 안전을 보장하는 지능형/능동형 스마트 차일드락 시스템**

본 프로젝트는 단순한 기계적 도어 잠금을 넘어, 센서 연동, 생체 신호 감지, 위치 기반 제어 등을 결합하여 차량 내외부의 다양한 위험 상황으로부터 어린이를 안전하게 보호하기 위해 기획된 통합 소프트웨어/하드웨어 제어 솔루션입니다.

---

## 📖 주요 기능 (Use Cases)

차량 탑승부터 하차, 그리고 비상 상황까지 전 주기를 커버하는 8가지 핵심 기능을 제공합니다.

* **UC-01: 지능형 하차 보조 (Advanced SEA)**
    * 하차 시 후측방에서 차량/이륜차 접근 감지 시 운전자가 해제를 시도해도 시스템이 물리적 잠금을 강제 유지하여 개문 사고 예방.
* **UC-02: 생체 신호 연동 비상 대응 (Active Rescue)**
    * 시동 OFF 후 차량 내 방치된 아동의 고체온/이상 호흡 감지 시, 창문 일부 개방(환기), 비상등/경적 작동 및 외부 구조자용 QR/NFC 활성화.
* **UC-03: 승객 구분형 스마트 잠금 (Smart Lock)**
    * 시트 무게 센서 및 실내 카메라를 통해 탑승객이 어린이(또는 카시트)로 판단될 경우 차일드락 자동 활성화.
* **UC-04: 크래시 언락 (Crash Unlock)**
    * 사고(충격 및 에어백 전개) 발생 시 배터리 상태와 무관하게 즉각적으로 모든 도어의 기계적 잠금 해제.
* **UC-05: 세이프티 리버싱 (Anti-Pinch)**
    * 윈도우 조작 중 신체 끼임(모터 부하) 감지 시 즉시 역회전(Reverse)하여 부상 방지.
* **UC-06: 하이브리드 제어 및 상태 시각화 (Hybrid Control & Visibility)**
    * 물리 버튼(일괄 제어)과 인포테인먼트 디스플레이(개별 제어)를 통한 유연한 제어 및 직관적 상태 시각화(LED/HUD 연동).
* **UC-07: 위치 기반 자동 해제 (Geo-Fencing Unlock)**
    * 사전 등록된 안전 구역(집, 학교 등) 도착 및 주차(P단) 시 자동으로 차일드락 해제.
* **UC-08: 실내 조작 감지 및 알림 (Internal Attempt Alert)**
    * 잠금 상태에서 뒷좌석 도어/창문 조작 시도 감지 시 운전자에게 즉시 클러스터 팝업 및 경고 알림.

## 🛡 안전 및 예외 대응 설계 (Safety & Fail-Safe Design)

어떠한 고장(Failure) 상황에서도 탑승자의 안전을 보장하기 위해 다음과 같은 안전 메커니즘이 적용되어 있습니다.

| 상황 | 위험 요소 | 대응 방안 (Safety Mechanism) |
| :--- | :--- | :--- |
| **전원 상실** | 차량 방전 시 기능 마비 | 비상 백업 배터리, 에어백 제어기 하드와이어링 연동, 전원 차단 시 기계적 락 유지 (Bistable Relay) |
| **센서 고장** | 센서 오작동으로 인한 오판 | 센서 진단 에러 시 가장 안전한 상태인 **Safe State (잠금 유지)**로 진입 및 클러스터 경고 |
| **비정상 조작** | 주행 중 버튼 오조작 | 속도 센서(3km/h 초과) 및 기어 상태 연동하여 주행 중 해제 명령 완전 무시 (인터락 로직) |
| **외부 간섭** | 자동 해제 시 후측방 위험 | GPS 안전 구역 도착으로 해제 명령이 내려져도 SEA 센서가 위험 감지 시 해제 명령 Override (보류) |

## 🛠 기술 스택 및 구조 (Tech Stack)

* **Language:** C (C99 / C11)
* **Testing & CI:** CTest 기반 단위 테스트 (Coverage 100% 목표), GitHub Actions
* **Architecture:**
    * `child_lock_system.h`: 하이브리드 제어 통합 인터페이스
    * `advanced_sea.h`: 지능형 하차 보조 제어 로직
    * `active_rescue.c / .h`: 생체 신호 연동 능동 구조 모드 시뮬레이션

## ⚙️ 빌드 및 실행 방법 (Getting Started)

본 프로젝트는 C 언어 환경에서 핵심 비즈니스 로직(Active Rescue 모드 등)을 시뮬레이션하고 테스트할 수 있도록 구성되어 있습니다. 윈도우(Windows) 및 리눅스(Linux) 환경을 모두 지원합니다.

$git clone [https://github.com/nohjinyoung/MBP-T03-ChildLock.git$](https://github.com/nohjinyoung/MBP-T03-ChildLock.git$) cd MBP-T03-ChildLock

$ ./active_rescue_sim
